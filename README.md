**This PHP library enables you to scrape data from IMDB.com.**

## Example Usage

```php
<?php
include_once 'imdb.class.php';
$IMDB = new IMDB('Movie Title or IMDB URL');
if ($IMDB->isReady) {
    print_r($IMDB->getAll());
} else {
    echo 'Movie not found. üòû';
}
```
